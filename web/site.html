A fullscreen User Interface for VLC

JuceVLC is inspired by fullscreen Mediaplayers like XBMC to provide an alternative the the built in VLC user interface.

The primary goal of this application is to provide a simple (and child frendly) player to select and watch movies from your couch using only a wireless mouse.

  • No system dialogs: browse files and configure player using On Screen Display
  • Mouse-only controls (Only one keboard shortcut: "space-key" to play/pause)
  • Easy audio / subtitles manual synchronization using slider
  • Simple favorite folders management
